# README

In my search for a favorite backend framework I've decided to take a look at Ruby On Rails.

A few considerations that I took into account:
- Popularity
    - Many companies use this therefore it's a good investment of time
- Usability
    - Seems like a very friendly framework (as well as Ruby itself being an easy language)
- Viability
    - I am, so far, discontent to use Django, Flask, Laravel, or any NodeJS based framework for the backend of my personal projects. Not for any particular reason, just don't like them that much. Rails seems like it could fit the bill, but I have to play with it first.
